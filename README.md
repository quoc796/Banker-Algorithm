# Banker-Algorithm
Simple implementation of Banker Algorithm using C
Bank Algorithms that allows upto 10 processes and 10 types of Resources
input.txt defines the input format 

Banker-Algorithm is algorithm designed evaluate the allocation of resources.
It allows the system to know the safest order of execution of requests.
How To use:
  Put input.txt and banker.c in a folder
  modify input.txt to fit your need
  compile banker.c
Documentation and example use provided
